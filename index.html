Diseñando Regiones

Software
POO
WindowBuilder
Trabajo Grupal

El software solicitado es una aplicación para resolver el problema de diseño de regiones de un país. Esta aplicación trabajará con un grafo que representa los distintos puntos en un mapa, donde cada vértice es una ubicación o provincia y cada arista representa la conexión entre esas ubicaciones, con un peso asociado que indica la similitud entre ellas.

La aplicación debe tener una interfaz para cargar y visualizar el grafo con sus vértices, aristas y pesos, así como una interfaz para mostrar las regiones resultantes después de aplicar el algoritmo. No es obligatorio tener una visualización gráfica o sobre un mapa, pero se pueden considerar elementos opcionales como estadísticas de las regiones, visualización en un mapa y la capacidad de recalcular las regiones al modificar el grafo.

2048

Juego
POO
WindowBuilder
Trabajo Grupal

El software solicitado es una aplicación de escritorio diseñada para ofrecer funcionalidades específicas relacionadas con un juego en una grilla de 4x4 donde se mueven números naturales. En cada turno, el jugador usa las flechas del cursor para mover los números en la dirección elegida. Se genera un nuevo número (2 o 4) en cada turno. Los números se mueven en la dirección seleccionada hasta chocar con otro número o el borde de la grilla. Cuando dos números iguales colisionan, se combinan sumándose y no se combinan más en ese turno. La aplicación proporcionará una interfaz intuitiva y fácil de usar para los usuarios.
